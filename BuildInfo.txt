3. Set SemiEXP > Properties > General > Configuration Type = Static Library
4. Set SemiExp > Properties > C/C++ > Preprocessor > TEST_SEMIEXP = zTEST_SEMIEXP
5. rebuild Tokenizer project
6. rebuild SemiExp project
